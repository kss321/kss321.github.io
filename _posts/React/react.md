    - 개발환경

3. 개발환경   
    1) Create React App 설치 순서
        - npm 설치 (node가 설치되어 있으면 됨)
        - Create React App 설치
            // 설치 
            npm install -g create-react-app     
            // 설치 버전 확인
            create-react-app -V
            // 4.0.1
        - 개발환경 완성    
    2) react start
        - npm run start     
    3) build 
        - npm run build
    4) deploy
        - npx serve -s build
